# 🧠 Digital Twin of Human

This project demonstrates a Digital Twin model of a human body using AI.

## 🚀 Features
- Health monitoring simulation (Heart Rate, SpO2, Motion)
- AI-based risk detection
- Future health prediction
- Visual dashboard (Graph + Gauge)
- Voice interaction (demo)

## 🧩 Tech Stack
- Python
- Gradio
- Scikit-learn
- Plotly

## 🔄 System Flow
Sensors → AI Model → Digital Twin → Prediction
